## CSS 变量

| 属性 | 说明 | 默认值
| - | - | -
| --wui-badge-font-size | 文字大小 | 9px
| --wui-badge-dot-size | 红点大小 | 10px
| --wui-badge-color | 颜色 | #ff3141
| --wui-badge-text-color | 文字颜色 | #fff
