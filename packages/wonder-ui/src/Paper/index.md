# Paper 内容墙

内容墙, 白色背景容器, 出现在灰色背景板上以突出内容.

阴影提供了关于物体深度和方向运动的重要视觉线索。他们是唯一的视觉线索，表明分离的数量之间的表面

## CSS 变量

| 属性 | 说明 | 默认值
| - | - | -
| --wui-paper-border-radius | 圆角 | 0px
| --wui-paper-elevation | 阴影, 通过`elevation`预设, 也可以手动修改 | -
