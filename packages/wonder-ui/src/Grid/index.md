## CSS 变量

| 属性 | 说明 | 默认值
| - | - | -
| --wui-grid-gap | 间距大小 |
| --wui-grid-gap-horizontal | 水平方向间距大小 | var(--wui-grid-gap)
| --wui-grid-gap-vertical | 垂直方向间距大小 | var(--wui-grid-gap)
