# Image 图片

增强版的 img 标签，提供多种图片填充模式，支持图片懒加载、加载中提示、加载失败提示。

## CSS 变量

| 属性 | 说明 | 默认值
| - | - | -
| --wui-image-height | 图片高度 | auto
| --wui-image-object-fit | 填充模式 | cover
| --wui-image-rounded-radius | 图标圆角 | `theme.shape.borderRadius`
| --wui-image-width | 图片宽度 | auto
