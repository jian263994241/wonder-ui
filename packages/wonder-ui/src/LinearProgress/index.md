### LinearProgress

除了支持 `HTMLElement` 所有属性, 还支持以下属性:

| 参数	|说明	|类型	|默认值
| --- | --- | --- | ---
| animated | 条纹动画 | boolean |
| classes | css |  |
| color | 颜色 | 'primary' \| 'secondary' \| 'success' \| 'error' \| 'danger' \| 'warning' \| 'info' | primary
| value | 值 | number |
| variant | 类型 | 'determinate' \| 'indeterminate' | indeterminate
