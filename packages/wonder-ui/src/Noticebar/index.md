## CSS 变量

| 属性 | 说明 | 默认值
| - | - | -
| --noticebar-height |高度| 40px
| --noticebar-padding-horizontal | 水平内边距 | 12px
| --noticebar-paddign-vertical | 垂直内边距 | 8px
| --noticebar-text-color | 文字颜色 | #ff6010
| --noticebar-bg-color | 背景颜色 | #fff9ed
| --noticebar-border-color | 边框颜色 | #fff3e9
