### Noticebar


| 参数	|说明	|类型	|默认值
| --- | --- | --- | ---
| actions | 操作区域 | ReactNode |
| classes | | |
| closable | 可关闭的 | boolean | false
| icon | 图标 | ReactNode |
| scrollable | 滚动 | boolean | false
| type | 类型 | 'success' \| 'info' \| 'warning' \| 'error' | info
| wrap | 换行 | boolean | false
