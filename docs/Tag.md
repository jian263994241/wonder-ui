

### 实心标签

```jsx
import { Tag, Flex } from '@wonder-ui/core';

<Flex>
  <div>
    <Tag>默认标签</Tag>
  </div>
  <div>
    <Tag color="primary">primary</Tag>
  </div>
  <div>
    <Tag color="secondary">secondary</Tag>
  </div>
  <div>
    <Tag color="#5576F0">#5576F0</Tag>
  </div>
</Flex>

```
